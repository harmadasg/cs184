# Berkeley's CS 184:  Foundations of Computer Graphics

In my ongoing journey to explore the recommended courses on [teachyourselfcs.com](https://teachyourselfcs.com), my next milestone is to delve into the captivating world of Computer Graphics. This venture involves immersing myself in the concepts and techniques that bring digital images and visual simulations to life. I'm using the following materials:
* The book Fundamentals of Computer Graphics
* The [online lectures](https://inst.eecs.berkeley.edu//~cs184/fa12/onlinelectures.html "online lectures") provided for the Fall 2012 Berkeley course

All the required assigments and resources to complete the program are availalbe at the [course website](https://inst.eecs.berkeley.edu//~cs184/fa12/assignments.html "course website").
